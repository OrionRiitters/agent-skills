# Agent Skills

This repository collects reusable agent skills for coding agents and AI assistants.

Skills are organized by source or owner:

- `anthropic/` — skills mirrored from [`anthropics/skills`](https://github.com/anthropics/skills)
- `pi/` — skills mirrored from [`badlogic/pi-skills`](https://github.com/badlogic/pi-skills)
- `orion/` — Orion's custom skills

The goal is to provide a single place to store, review, and share skill definitions, helper scripts, references, and supporting assets that can be installed into agent environments.

When adding third-party skills, keep attribution, license files, and source metadata with the imported content.
